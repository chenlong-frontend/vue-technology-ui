## 空提示

### 基础用法

:::demo
```html
<k-empty description="没有更多数据了"></k-empty>
```
:::

### empty Attributes
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| image | 图片地址 | string | - | - |
| description | 文字描述 | string | - | - |
